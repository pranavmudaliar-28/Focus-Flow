Features

Everything you need to focus

Built for individuals who want deep work habits, and teams who need consistent, measurable productivity.

Focus Timer

Customisable sessions (15–240 min) with a smooth SVG ring. Chrome Alarms API keeps it running even after you close the popup.

Ambient Sounds

9 procedurally generated soundscapes — Rain, Forest, Café, Ocean, Fireplace, Brown Noise and more. Plays in a pinned background tab.

Site Blocker

Blocks Twitter, Reddit, TikTok and more during sessions. Admin org policies lock sites employees cannot override.

Streaks & Stats

Daily streaks, 7-day activity heatmap, and full session history. Stay consistent and track your momentum.

Org Mode

Join your team with a 6-character org code. Admin policies — blocklist, schedule, announcements — sync automatically.

Team Analytics

Admins see focus hours, compliance rates, and a leaderboard for the whole team. Export to CSV anytime.

Installation guide

Detailed setup steps

Full walkthrough for getting started — individual or team.

01

Install the extension

Install from the Chrome Web Store, or load manually: open chrome://extensions, enable Developer mode, click Load unpacked, and select the Focus Flow folder.

02

Pin it to your toolbar

Click the puzzle piece icon in Chrome → find Focus Flow → click the 📌 pin. It's always one click away from your toolbar.

03

Sign in (optional)

Log in with email or Google to enable streak sync across devices. Skip this step for a fully local, no-account experience — all features still work.

04

Start your first focus session

Name your task → pick a duration → hit Start Focus Session. Distraction sites are blocked automatically. Try an ambient sound from the Sounds tab.

05

Join your organisation Optional · Team

Go to the Org tab → enter the 6-character code from your admin → click Join Organisation. Admin policies sync instantly.

06

Set up as an admin Optional · Admin

Visit the Focus Flow Admin Panel → create an org → invite your team with the org code → manage blocklist, push announcements, and view team analytics.
